# agent: 금고 전문점 단일 페이지 웹사이트 프로젝트

## 0. 답변은 한국어 사용
## 1. 프로젝트 개요

- 목적: 아버지가 30년 동안 운영해 온 **금고 전문점**의 새로운 홈페이지를 제작한다.
- 핵심 서비스(항상 강조해야 할 키워드):
  - **중고 금고 판매**
  - **금고 수리**
  - **금고 운반**
- 목표:
  - 방문자가 “30년 경력 금고 전문점”이라는 **신뢰감**을 느끼게 하기
  - **전화 / 카카오톡 / 문의**로 이어지도록 하는 **전단지 스타일의 실전형 웹페이지** 제작

## 2. 기술 스택 및 기본 구조

- 기본 기술:
  - **HTML5**: 단일 페이지 구조
  - **CSS3**: 레이아웃 및 디자인
  - **JavaScript(선택)**: 상단 메뉴 클릭 시 부드러운 스크롤, 간단한 인터랙션 정도
- 파일 구조(기본):

  ```text
  프로젝트_폴더/
   ├─ index.html       // 메인 단일 페이지
   ├─ style.css        // 전체 스타일
   └─ images/          // 금고, 작업 사진 등
        └─ (이미지 파일들)
페이지 형태:

한 페이지 스크롤형(One Page) 웹사이트

상단 네비게이션 메뉴 클릭 → 같은 페이지 내 섹션으로 스크롤 이동
(a 태그의 href="#section-id" 방식 사용)

## 3. 레이아웃 설계 (섹션 구조)
페이지는 위에서 아래로 다음 순서의 섹션으로 구성한다.

header : 로고 + 상단 메뉴(네비게이션 바)

section#hero : 메인(Hero) 영역 – 3대 서비스 키워드 한눈에

section#services : 중고 금고 판매 / 금고 수리 / 금고 운반 요약 카드 3개

section#used-safe : 중고 금고 판매 상세 안내

section#repair : 금고 수리 상세 안내

section#moving : 금고 운반 상세 안내

section#about : 30년 경력, 장인 소개, 스토리

section#cases : 작업 사례 / 사진 / 후기

section#process : 이용 절차(문의 → 상담 → 작업 → 완료)

section#contact : 전화, 카톡, 주소, 영업시간, 위치 안내

footer : 간단한 저작권, 상호명, 연락처 재노출

3.1 header (상단 네비게이션)
구성:

왼쪽: 로고 또는 상호명 텍스트 (예: OO 금고 또는 OO 금고 30년)

오른쪽: 메뉴 목록 (각 섹션 id와 연결)

회사소개 → #about

중고 금고 판매 → #used-safe

금고 수리 → #repair

금고 운반 → #moving

이용 안내 / 문의 → #contact

요구사항:

상단에 고정된(고정 헤더) 형태로 구현하면 좋음 (position: fixed)

모바일에서는 햄버거 메뉴로 단순히 접거나, 메뉴를 세로로 나열해도 됨

3.2 section#hero (메인 Hero 영역)
목적:

첫 화면에서 바로 세 가지 핵심 키워드와 30년 경력을 보여준다.

방문자가 “여기는 진짜 금고 전문점이구나”라고 직관적으로 이해하도록 한다.

구성:

큰 제목(H1)

짧은 설명 문단 1~2개

주요 서비스 키워드 문장에 중고 금고 판매 / 금고 수리 / 금고 운반 포함

행동 유도 버튼(CTA) 2~3개

전화 상담

카카오톡 상담

견적 문의

예시 문구(참고용, 실제 HTML에 그대로 써도 됨):

text
코드 복사
[H1] 30년 경력 금고 전문점 – 중고 금고 판매 · 금고 수리 · 금고 운반

가정용부터 업소용까지, 믿고 맡길 수 있는 동네 금고 장인입니다.
급한 금고 고장, 문 안 열림, 금고 이전 설치까지 한 번에 해결해 드립니다.
3.3 section#services (서비스 요약 카드 3개)
목적:

세 가지 핵심 서비스를 한 번에 시각적으로 인지하게 만든다.

구성:

3개의 카드/박스를 가로 또는 세로로 배치

중고 금고 판매

금고 수리

금고 운반

각 카드에는:

작은 아이콘 또는 심플한 이미지(선택)

서비스 명(H3)

짧은 설명 2~3줄

예시:

중고 금고 판매: “상태 좋은 중고 금고를 새 제품보다 저렴하게 제공”

금고 수리: “문이 안 열리거나 비밀번호를 잊었을 때 현장 출동 수리”

금고 운반: “이사, 매장 이전 시 안전한 금고 이동 및 설치”

3.4 section#used-safe (중고 금고 판매 상세)
목적:

“중고 금고 판매” 키워드를 중심으로 상세 설명

중고 금고를 찾는 고객에게 신뢰 + 구체적인 정보 제공

구성:

제목(H2): 중고 금고 판매 – 새 것 같은 상태, 합리적인 가격

설명 문단 2~4개

30년 장사를 하며 중고 금고를 함께 취급해 온 점

상태가 좋은 금고만 선별하여 정비 후 판매한다는 점

가정용/사무실/업소 용도별 추천 가능

작은 표 또는 리스트:

용도(가정/사무실/상가 등)

크기(소/중/대)

대략적인 가격대(구체 가격은 문의 유도)

실제 중고 금고 사진(있다면 images/used-safe-01.jpg 등 사용)

3.5 section#repair (금고 수리 상세)
목적:

“금고 수리” 키워드를 강조하고, 자주 발생하는 문제를 적어 공감 유도

구성:

제목(H2): 금고 수리 – 갑자기 안 열릴 때, 당황하지 마세요

문단: 30년 경력, 직접 현장 방문 수리, 브랜드·모델 무관 가능 등 설명

자주 발생하는 고장 유형 리스트(ul/li):

비밀번호를 잊어버린 경우

손잡이가 돌아가는데 문이 안 열리는 경우

열쇠 분실 또는 파손

오래되어 잠금 장치가 뻑뻑해진 경우

사진 또는 아이콘은 선택사항

“사진을 보내면 대략 견적 안내 가능” 같은 문구 포함

3.6 section#moving (금고 운반 상세)
목적:

“금고 운반” 키워드 강조

금고가 무거워서 직접 옮기기 어려운 점을 짚어주고, 전문 운반의 필요성 강조

구성:

제목(H2): 금고 운반 – 이사, 매장 이전도 걱정 없이

문단: 금고 무게, 바닥 손상, 인명 사고 위험 등을 언급하면서 전문 운반 장점 설명

활용 사례 리스트:

가정 이사 중 금고 이전

사무실·상가 리모델링 시 금고 위치 변경

건물 층간 이동(엘리베이터 없음, 계단 운반)

“현장 상황(층수, 계단 유무, 금고 크기)을 미리 물어보고 준비한다”는 내용 포함

3.7 section#about (30년 경력, 장인 소개)
목적:

**“30년 경력 금고 장인”**이라는 신뢰와 스토리 전달

구성:

제목(H2): 30년 금고 장인의 손길

문단: 한 자리에서 30년 장사, 고객의 자산을 지키는 마음가짐, 정직한 견적 등

아버지의 작업 모습 사진이 있다면 넣기 (선택)

“대형 업체와 차별되는 개인 장인의 책임감” 같은 메시지 포함

3.8 section#cases (작업 사례 / 후기)
목적:

실제 사례와 사진으로 신뢰도 상승

구성:

제목(H2): 작업 사례 및 후기

카드 형태 또는 리스트:

“○○동 사무실 금고 수리 – 문 안 열리는 고장” (사진 1~2장)

“△△아파트 가정용 금고 운반 – 3층 계단 운반”

“□□상가 중고 금고 판매 및 설치”

실제 후기 텍스트가 있다면 짧게 인용

3.9 section#process (이용 절차)
목적:

방문자가 “어떻게 진행되는지”를 이해하게 돕는다.

구성:

제목(H2): 이용 절차

4단계 정도로 시각화:

문의 (전화 / 카카오톡)

사진 및 상황 안내 → 대략적인 견적 안내

방문 후 작업 진행 (판매/수리/운반)

마무리 및 결제

각 단계에 아이콘 또는 숫자 배지 사용

3.10 section#contact (문의 / 연락처 / 위치)
목적:

이 페이지의 최종 행동 유도 영역

구성:

제목(H2): 문의 및 연락처

필수 정보:

전화번호 (클릭 시 통화 연결 가능하도록 tel: 링크 사용)

카카오톡 ID 또는 채널 링크

주소(도시/구/도로명 정도)

영업시간

지도 삽입은 나중 단계(구글맵, 카카오맵 iframe 등)로 선택

예:

📞 전화: 010-XXXX-XXXX

💬 카카오톡: OO금고

📍 주소: ○○시 ○○구 ○○로 123, 1층

🕒 영업시간: 평일 09:00 ~ 19:00 / 주말 문의 가능

5. 반응형(모바일) 기본 요구사항
필수 수준:

모바일(세로 화면)에서 내용이 잘려 보이지 않고, 세로 스크롤만으로 모두 읽을 수 있어야 한다.

상단 메뉴는 줄이 좁아지면 세로로 쌓이거나, 단순한 햄버거 버튼으로 변경해도 된다.

구현 방식:

@media 쿼리를 사용하여 768px 이하 화면에서 카드 3개를 세로로 배치

글자 크기와 버튼 크기를 터치하기 편한 정도로 조정

## 6. SEO(검색 최적화) 기본 요구사항
<title>:

예시: OO금고 – 중고 금고 판매 · 금고 수리 · 금고 운반 (30년 경력 금고 전문점)

<meta name="description">:

예시:
30년 경력 금고 전문점 OO금고. 중고 금고 판매, 금고 수리, 금고 운반까지 한 번에 해결해 드립니다. 가정용, 사무실, 업소용 금고 상담 환영.

키워드 배치:

H1, 각 섹션 H2에 다음 문구 자연스럽게 포함

“중고 금고 판매”

“금고 수리”

“금고 운반”

본문에도 서비스 설명 중 자연스럽게 3~5회 정도 반복
