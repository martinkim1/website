# 금고백화점 (GoldSafe) 웹사이트

## 1. 개요 및 목적
30년 경력의 금고 전문점 '금고백화점'의 온라인 홍보를 위한 단일 페이지 웹사이트입니다.
- **주요 서비스**: 중고 금고 판매, 금고 수리, 금고 운반
- **목적**: 신뢰감 있는 디자인과 직관적인 정보 전달을 통해 전화 및 문자 문의를 유도합니다.

## 2. 기술 스택
- **HTML5**: 시멘틱 마크업 구조
- **CSS3**: 반응형 디자인, CSS Animation (Hero 슬라이드쇼), Flexbox 레이아웃
- **JavaScript**: 없음 (순수 CSS로 구현하여 유지보수 용이성 확보)

## 3. 설치 방법
이 프로젝트는 별도의 빌드 과정이나 설치가 필요 없습니다.
1. 소스 코드를 다운로드합니다.
2. `site` 폴더가 웹 서버의 루트(Root) 역할을 합니다.

## 4. 실행 방법
### 로컬 실행
- `site/index.html` 파일을 크롬 등 웹 브라우저로 열면 바로 실행됩니다.
- 또는 VS Code의 'Live Server' 확장 기능을 사용하면 더 원활하 테스트가 가능합니다.

### 배포 실행
- `site` 폴더 전체를 정적 웹 호스팅 서비스(Netlify, Vercel, Github Pages 등)에 업로드하면 바로 서비스가 가능합니다.

## 5. 제공 기능
- **메인 Hero 슬라이드쇼**: CSS Keyframes를 활용한 이미지 자동 전환 효과
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모두에 최적화된 레이아웃
- **상단 고정 헤더**: 스크롤 시에도 항상 노출되는 네비게이션
- **원페이지 스크롤**: 메뉴 클릭 시 해당 섹션으로 부드럽게 이동
- **연락처 연결**: 전화 걸기(`tel:`) 및 문자 보내기 기능 내장
